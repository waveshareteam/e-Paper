    # modle is 7.5inch e-Paper (B) (V2)
    sudo make clean
    sudo make EPD=epd7in5bV2

编译的时候加上 EPD define，例如：
    # 屏幕型号为 7.5inch e-Paper (B) (V2)
    sudo make clean
    sudo make EPD=epd7in5bV2
