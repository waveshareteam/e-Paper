使用方法：
1. 在 Arduino IDE 中安裝 Ameba Arduino SDK。 GitHub: https://github.com/Ameba-AIoT/ameba-arduino-pro2
2. 前往 Realtek_AmebaAIoT -> examples，並選擇與您使用的電子紙尺寸相符的資料夾名稱。
3. 開啟相應的 .ino 檔案。
4. 在 Arduino IDE 中選擇 Ameba 開發板。
5. 編譯並上傳。

或者，您也可以將整個資料夾（例如 “AmebaD”）壓縮成 .zip 檔，然後在 Arduino IDE 中導航到 Sketch" -> "Include Library" -> "Add .ZIP Library..." 加入.
添加後，您可以在 "Files" -> "Examples" -> "AmebaEink" 中看到所有的範例。

English：
Supported Boards: 
- AMB21/22 (AmebaD)
- AMB23 (AmebaD)
- AMB25 (AmebaD)
- AMB26 (AmebaD)
- BW16 (AmebaD)
- BW16 Type C (AmebaD)
- AW-CU488 (AmebaD)
- AMB82 Mini (AmebaPro2)

Supported E-Paper:
- 2.9inch/2.9inch v2/ 2.9inch (B)
- 4.2 inch/ 4.2 inch v2
- 7.5 inch

How to use:
1. Install Ameba Arduino SDK in Arduino IDE.
GitHub: https://github.com/Ameba-AIoT/ameba-arduino-pro2
2. Go to Realtek_AmebaAIoT -> examples and choose the folder name with the size of the e-Paper that you are using.
3. Open respective .ino file.
4. Choose Ameba boards in Arduino IDE.
5. Compile and Upload.

Alternatively, you can also zip the whole folder e.g. "AmebaD" and add the .zip folder in Arduino IDE by naviagting to "Sketch" -> "Include Library" -> "Add .ZIP Library...".
After adding, you will see all the examples in "Files" -> "Examples" -> "AmebaEink".

Example Documentation: https://ameba-doc-arduino-sdk.readthedocs-hosted.com/en/latest/index.html
